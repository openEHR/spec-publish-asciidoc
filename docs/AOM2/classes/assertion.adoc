{set:cellbgcolor!}

{set:cellbgcolor:lightblue}

|*0..1*
{set:cellbgcolor:lightblue}
|*string_expression*: 
{set:cellbgcolor!}

|*0..1*
{set:cellbgcolor:lightblue}

{set:cellbgcolor!}
|Definitions of variables used in the assertion expression.

|*1..1*
{set:cellbgcolor:lightblue}
|*expression*: `EXPR_ITEM`
{set:cellbgcolor!}
